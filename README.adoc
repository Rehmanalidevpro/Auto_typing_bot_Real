== System Overview

Auto Typing Bot Real is built for professionals who require automated text injection without triggering automated security flags. The engine utilizes complex randomization to mimic Words Per Minute (WPM) and human psychological typing patterns. 

It comes with an integrated command-line interface (CLI) that provides real-time monitoring and advanced configuration wizards.

== Core Features

* *Anti-Detection Engine:* Implements dynamic micro-delays between keystrokes (0.04s to 0.15s by default) to simulate physical keyboard hardware interrupts.
* *Smart Pause & Resume Mechanism:* The system actively monitors global mouse events. Clicking anywhere outside the target input box immediately pauses the typing sequence. Clicking back inside the target box resumes the sequence seamlessly.
* *Natural Typo Injection:* Intentionally generates organic typing mistakes and automatically corrects them using the Backspace key, mimicking human error rates.
* *Psychological Pauses:* Simulates human "thinking breaks" (1 to 5 seconds) randomly at the end of words or sentences.
* *Automated Dependency Management:* The script automatically scans your environment and installs missing required libraries via PIP upon initial execution.
* *Custom Dictionary Mapping:* Auto-generates a `words.json` file, allowing users to define custom text shortcuts (e.g., auto-expanding "i am" to "I'm").

== System Requirements

* Python 3.8 or higher
* Active internet connection (only for the first run to download dependencies)
* Supported OS: Windows, macOS, Linux

== Installation & Execution

You do not need to manually install dependencies. The system handles its own environment setup.[source,bash]
----
# Clone the repository to your local machine
git clone https://github.com/Rehmanalidevpro/Auto_typing_bot_Real.git

# Navigate into the directory
cd Auto_typing_bot_Real

# Execute the primary script
python typing_bot.py
----

== Usage Instructions

1. Launch the application using `python typing_bot.py`.
2. From the Main Menu, select option `1` to Initialize the Auto Typing Engine.
3. Paste your large block of text into the terminal (using `Ctrl+V` or Right-Click).
4. On a new blank line, type the word `DONE` and press `Enter`.
5. You will be given a 4 to 10-second initialization window. 
6. *Action Required:* Immediately click inside your target text box (browser, document, etc.).
7. The system will take control and begin the simulated typing sequence.

[NOTE]
====
*Fail-Safe Mechanism:* If you need to force-stop the bot immediately, move your physical mouse cursor to any of the four extreme corners of your screen. The system will trigger an emergency halt.
====

== Configuration System

The tool generates a `config.json` file in the root directory. You can modify these parameters either directly in the JSON file or via the CLI Settings Menu (Option 2).

* *Speed Configuration:* Adjust minimum and maximum keystroke delays. Includes presets (Slow, Normal, Fast) and Custom parameters.
* *Typo Probability:* Adjust the percentage chance of the system making a deliberate typing error.
* *Pause Probability:* Configure how frequently the bot should take a simulated thinking break.
